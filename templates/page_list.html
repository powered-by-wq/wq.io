wq on GitHub

wq is a modular framework for building custom offline-capable desktop and mobile web apps for volunteered geographic information, citizen science, and crowdsourcing.

wq is open source and extends a number of existing open source web frameworks, including Django, jQuery Mobile, and RequireJS.

wq 0.6.1 was released on July 8th, 2014!
